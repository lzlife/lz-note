# 📝 更新日志

## 1.0.0 (2025-07-10)

### 🎯 核心功能

#### 编辑器
- **所见即所得编辑**：基于 Vditor WYSIWYG 模式，支持完整的 Markdown 编辑体验
- **富文本工具栏**：撤销/重做、标题、粗体/斜体/删除线、分割线、引用、无序/有序/任务列表、代码/行内代码、表情、图片上传、链接、表格、编辑模式切换、大纲、全屏
- **斜杠命令菜单**：输入 `/` 触发快捷命令面板，支持标题2-6、代码块、引用、列表、链接、表格等 13 种插入
  - 键盘方向键/回车/Esc 导航
  - 实时输入过滤
  - 视口边界自动检测，超出窗口范围自动偏移
  - 点击面板外部自动关闭
  - 删除 `/` 后菜单立即消失（无延迟）
- **图片上传**：拖拽/粘贴/工具栏上传图片，自动保存至文件同目录的 `.resources/` 文件夹
- **自动保存草稿**：编辑内容实时写入 localStorage，切换文件时 400ms 防抖落盘
- **草稿恢复**：异常退出后可自动恢复未保存内容
- **深色/浅色主题**：跟随系统主题自动切换编辑器配色

#### 文件管理
- **侧边栏文件树**：递归展示工作区目录结构
- **文件操作**：新建笔记、新建文件夹、重命名（行内编辑）、拖拽移动、删除确认
- **搜索过滤**：按文件名实时过滤文件树
- **导入 Markdown**：支持导入现有 `.md` 文件到工作区

#### 导出
- **Markdown 导出**：保持原始格式
- **HTML 导出**：Vditor 引擎渲染为完整 HTML 文档
- **PDF 导出**：通过 Playwright 渲染为 PDF
- **图片导出**：通过 Playwright 渲染为 PNG/JPG
- **文件夹导出**：直接复制整个目录

#### Git 同步
- **完整同步工作流**：预检查 → 拉取 → 差异比对 → 提交 → 推送
- **三种同步策略**：全量同步 / 仅拉取 / 手动
- **冲突处理**：快照对比 + 用户决策界面
- **启动自动同步**：应用启动时按策略自动执行
- **提交信息自动生成**：基于变更文件列表自动拼接

### ⚙️ 技术架构

- **框架**：React 19 + TypeScript 5
- **构建**：Vite 6
- **样式**：Tailwind CSS v4 + shadcn/ui
- **状态管理**：Zustand v5
- **编辑器内核**：Vditor v3.11.2（WYSIWYG 模式）
- **Git**：isomorphic-git（纯 JS 实现）
- **导出**：Playwright（子进程）
- **运行环境**：ZTools 插件宿主
